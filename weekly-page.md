<h1>Week 1: Class Introduction, Version Control and Basic HTML</h1>
<p>This week will introduce you to a <em>version control system</em> called Git. Version control systems let you more easily work with others on the same set of source code. Additionally it keeps a history so you can roll back changes. Finally it can keep track of different versions so you can have unfinished features that will not be in the production code but might exist in development code.</p>
<p>Coming out of this week you should have learned the following:</p>
